- run ./makesymlinks to link the files into ~/home/user/

- install vundle vim plugin manager from https://github.com/VundleVim/Vundle.Vim
[- install all vim plugins by launching vim and running :PluginInstall]
[- install YouCompleteMe fully by following https://github.com/Valloric/YouCompleteMe/blob/master/README.md#ubuntu-linux-x64-installation]
